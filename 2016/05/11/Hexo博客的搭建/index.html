Hexo博客的搭建

By 邹强

2016-05-11 更新日期:2016-05-12

大家一定在网上查过关于Hexo博客的搭建并部署到Github，网上写的千篇一律，今天来跟大家详细说一说，我搭建Hexo博客的过程！我决定写一篇“史上最详细“截图”搭建Hexo博客并部署到Github”。

工具/原料:

*  Mac

*  Hexo

*  Github

*  Node.js

Node.js 的下载/安装

下载

大家进入nodejs的官网进行下载。https://nodejs.org/en/
OSX系统下载左边的V4.4.4LTS稳定版，其他系统（例如：windows）点击下面的other Downloads进行选择下载！

安装

下载后，进行安装，如图所示，一路确认就行了！









Hexo 安装

安装Hexo之前，需要安装npm环境，npm官网：https://www.npmjs.com/。
1，打开终端





2，输入命令：
npm install -g hexo
注意：-g是指全局安装hexo





3，创建hexo文件夹
完成
