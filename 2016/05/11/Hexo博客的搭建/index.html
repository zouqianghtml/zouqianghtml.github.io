Hexo博客的搭建

By 邹强

2016-05-11 更新日期:2016-05-16

大家一定在网上查过关于Hexo博客的搭建并部署到Github，网上写的千篇一律，今天来跟大家详细说一说，我搭建Hexo博客的过程！我决定写一篇“史上最详细“截图”搭建Hexo博客并部署到Github”。

工具/原料:

*  Mac

*  Hexo

*  Github

*  Node.js

Node.js 的下载/安装

下载

大家进入nodejs的官网进行下载。https://nodejs.org/en/
OSX系统下载左边的V4.4.4LTS稳定版，其他系统（例如：windows）点击下面的other Downloads进行选择下载！

安装

下载后，进行安装，如图所示，一路确认就行了！





Hexo 安装

安装Hexo之前，需要安装npm环境，npm官网：https://www.npmjs.com/。
1，打开终端





2，输入命令：
npm install -g hexo
注意：-g是指全局安装hexo



3，创建hexo文件夹
这里我们在桌面建一个名为blog文件夹为例。
在终端输入cd ~Desktop/blog (~表示自己电脑桌面路径)进入blog文件夹,
然后输入hexo init

回车！等待hexo配置文件下载

下载完成后，打开blog文件夹，文件夹里面会生成如下所示的文件，那么现在恭喜你，你的本地hexo博客已经建好了，下面我们来写几篇博客吧！



4，写博客
输入hexo new 你的博客名称 新建一篇博客

回车！然后打开blog文件夹blog/source/_posts, _posts文件夹里面有里有刚才你新建的一篇博客。

在blog文件夹下面，输入hexo server命令,会跳出一个本地连接的网址，如下图。

在浏览器中输入这个链接，就会进入你的博客主页。如下图：

到此，你的本地博客就算建好了！


5，配置github
注册github，这里就展开讲了，自己去官网注册一个账号！https://github.com

首先我们需要新建一个Repository
建立与你用户名对应的仓库，仓库名必须为【your_user_name.github.io】,如下图：

然后复制这个库的HTTPS地址，下面需要用到。

打开的blog文件夹下面的_config.yml文件。

在blog文件夹下面输入npm install hexo-deployer-git --save命令

接着在blog文件夹下面输入hexo g hexo d 这两个命令。
最后你在浏览器输入zouqinaghtml.github.io这个网址就行了，我的github的账户叫zouqianghtml,把这个改成你github的账户名就行了



报错解决方法

ERROR Deployer not found: git 或者 ERROR Deployer not found: github
解决方法： npm install hexo-deployer-git –save

如发生报错： ERROR Process failed: layout/.DS_Store , 那么进入主题里面layout和_partial目录下，使用删除命令：
rm-rf.DS_Store
ERROR Plugin load failed: hexo-server
